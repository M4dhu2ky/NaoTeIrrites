# 🎲 Não Te Irrites! – Mobile Game (Multiplayer & Offline)

> 🌍 [English](#english) | [Português](#português)

---

## English

This repository contains the development of a digital adaptation of the classic board game **"Não Te Irrites!"**, one of the most beloved games in Angolan entertainment culture. The project focuses on bringing the strategic board experience to mobile, serving as a foundation for future cross-platform expansions.

---

### 📝 About the Project
Development focuses on faithfulness to traditional rules, combined with a robust C# code architecture. This project is my applied field for gameplay logic systems and responsive UI design.

### 🎮 Key Features
- **Free-for-All System (FFA):** Logic for 4 simultaneous players.
- **Game Modes:** Functional Offline (Local) version, with architecture prepared for Online Multiplayer.
- **"Rolha" Mechanic:** Implementation of the "Perfect Win" (when the winner finishes the game without any opponent placing a piece at home).
- **Visual Feedback:** Complete splash screen with particle systems, animations and immersive audio.
- **Login Screen:** Animated entry sequence with DOTween — logo drop, eye expressions and button fade-in.
- **Main Menu:** Icon-based navigation with animated entry transitions.

### 🛠️ Tech Stack
- **Engine:** Unity 6 LTS
- **Language:** C#
- **UI System:** Unity Canvas (mobile performance focused)
- **Animations:** DOTween v1.2.825
- **Version Control:** Git

---

### 🏗️ Cross-Platform Vision
Although the initial release is focused on mobile devices, the codebase is being developed in a modular fashion, allowing portability to **PC and Web** in the future, maintaining consistency in game logic and UI systems.

---

### 🚀 Current Status
This project is in active development.
> Currently focused on completing the offline version and polishing scene transitions.

**Roadmap:**
- [x] Splash Screen (Sanctum Nova intro)
- [x] Login Screen (animated sequence)
- [x] Main Menu (layout + entry animation)
- [ ] Game Scene (offline mode)
- [ ] Online Multiplayer
- [ ] Android Build (v1.0)

## Português

Este repositório contém o desenvolvimento de uma adaptação digital do clássico jogo de mesa **"Não Te Irrites!"**, um dos jogos mais queridos da cultura de entretenimento em Angola. O projeto foca em levar a experiência estratégica do tabuleiro físico para o ambiente mobile, servindo como base para futuras expansões multiplataforma.

---

### 📝 Sobre o Projeto
O desenvolvimento foca na fidelidade às regras tradicionais, aliada a uma arquitetura de código robusta em C#. Este projeto é o meu campo de aplicação para sistemas de lógica de gameplay e design de interface (UI) responsiva.

### 🎮 Principais Funcionalidades
- **Sistema Free-for-All (FFA):** Lógica para 4 jogadores em simultâneo.
- **Modos de Jogo:** Versão Offline (Local) funcional, com arquitetura preparada para Multiplayer Online.
- **Mecânica da "Rolha":** Implementação do "Perfect Win" (quando o vencedor termina o jogo sem que nenhum oponente coloque uma peça em casa).
- **Feedback Visual:** Splash screen completa com sistemas de partículas, animações e áudio imersivo.
- **Ecrã de Login:** Sequência de entrada animada com DOTween — queda do logo, expressões dos olhos e fade-in dos botões.
- **Menu Principal:** Navegação por ícones com transições de entrada animadas.

### 🛠️ Stack Tecnológica
- **Engine:** Unity 6 LTS
- **Linguagem:** C#
- **UI System:** Unity Canvas (focado em performance mobile)
- **Animações:** DOTween v1.2.825
- **Versionamento:** Git

---

### 🏗️ Visão Multiplataforma
Embora o lançamento inicial seja focado em dispositivos móveis, a arquitetura do código está a ser desenvolvida de forma modular, permitindo a portabilidade para **PC e Web** no futuro, mantendo a consistência da lógica de jogo e dos sistemas de UI.
