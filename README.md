# 🎲 Não Te Irrites! – Mobile Game (Multiplayer & Offline)

Este repositório contém o desenvolvimento de uma adaptação digital do clássico jogo de mesa **"Não Te Irrites!"**, um dos jogos mais queridos da cultura de entretenimento em Angola. O projeto foca em levar a experiência estratégica do tabuleiro físico para o ambiente mobile, servindo como base para futuras expansões multiplataforma.

---

## 📝 Sobre o Projeto
O desenvolvimento foca na fidelidade às regras tradicionais, aliada a uma arquitetura de código robusta em C#. Este projeto é o meu campo de aplicação para sistemas de lógica de gameplay e design de interface (UI) responsiva.

### 🎮 Principais Funcionalidades
- **Sistema Free-for-All (FFA):** Lógica para 4 jogadores em simultâneo.
- **Modos de Jogo:** Versão Offline (Local) funcional, com arquitetura preparada para Multiplayer Online.
- **Mecânica da "Rolha":** Implementação do "Perfect Win" (quando o vencedor termina o jogo sem que nenhum oponente coloque uma peça em casa).
- **Feedback Visual:** Splash screen completa com sistemas de partículas, animações e áudio imersivo.

## 🛠️ Stack Tecnológica
- **Engine:** Unity
- **Linguagem:** C#
- **UI System:** Unity Canvas (focado em performance mobile)
- **Animações:** DOTween (em fase de integração para suavidade de movimentos)
- **Versionamento:** Git

---

## 🏗️ Visão Multiplataforma
Embora o lançamento inicial seja focado em dispositivos móveis, a arquitetura do código está a ser desenvolvida de forma modular, permitindo a portabilidade para **PC e Web** no futuro, mantendo a consistência da lógica de jogo e dos sistemas de UI.
